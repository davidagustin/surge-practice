## 🚀 Quick Start

### Prerequisites

- Node.js (for Surge CLI)
- Git (for version control)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/davidagustin/surge-practice.git
   cd surge-practice
   ```

2. **Install Surge globally:**
   ```bash
   npm install --global surge
   ```

3. **Deploy to Surge:**
   ```bash
   surge
   ```

## 📖 Detailed Deployment Guide

### Step 1: Install Surge CLI

```bash
npm install --global surge
```

### Step 2: Deploy Your Site

```bash
# Navigate to your project directory
cd surge-practice

# Deploy to Surge
surge
```

### Step 3: Follow the Prompts

- Enter your email address
- Create a password or use existing account
- Choose a domain name (e.g., `my-awesome-site.surge.sh`)

### Step 4: Your Site is Live!

Your website will be available at: `https://your-chosen-domain.surge.sh`

## 🎨 Customization

### Colors and Theme

The website uses a beautiful purple/blue gradient theme. To customize:

1. **Primary Colors** - Edit the gradient in `styles.css`:
   ```css
   background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
   ```

2. **Text Colors** - Modify the color variables:
   ```css
   color: #333; /* Main text */
   color: #666; /* Secondary text */
   ```

### Content

- **Hero Section** - Edit the main title and subtitle in `index.html`
- **About Section** - Modify feature cards and descriptions
- **Contact Information** - Update contact details and form fields

### Styling

- **Typography** - Change fonts in `styles.css`
- **Layout** - Adjust spacing and grid layouts
- **Animations** - Modify CSS animations and transitions

## 🔧 Development

### Local Development

1. **Open in browser:**

2. **Use a local server:**
   ```bash

   python -m http.server 8000
   
   # Node.js
   npx serve .
   
   # PHP
   php -S localhost:8000
   ```

3. **Visit:** `http://localhost:8000`
